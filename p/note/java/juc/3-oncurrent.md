## 多线程通信

## 1.demo1

```
实现一个容器，提供两个方法，add，size
写两个线程，线程1添加10个元素到容器中
线程2实现监控元素个数，当个数到5个时，线程2提示并结束
```

## 2.demo2

```
写一个固定容量同步容器，有put，get，getCount方法，
能够支持2个生成线程，及10个消费者线程的阻塞调用
```
