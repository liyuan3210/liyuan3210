### 一．数据结构分类

#### 线性结构：

数组，队列，链表和栈

```
特点是数据元素之间存在一对一的线性关系

有两种不同存储结构，顺序存储（存储元素是连续的），链式存储（元素不一定是连续的，元素存储相邻的元素地址信息）
```
