## 一．数据结构与算法

应用场景　＝ 	数据结构	＋　算法

### １．数据结构分类

特点是数据元素之间存在一对一的线性关系

有两种不同存储结构，顺序存储（存储元素是连续的），链式存储（元素存储相邻的元素地址信息）

##### 线性结构：

数组，队列，链表和栈

#### 非线性结构：

二维数组，多维数组，广义表，树结构，图结构
